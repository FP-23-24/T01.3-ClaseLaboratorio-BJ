USER INSTRUCTIONS:

Es una adaptación del proyecto del capítulo 1. Hay dos versiones del proyecto: versión Eclipse y 
versión BlueJ.

El objetivo de este proyecto es ilustrar la interacción entre objetos y examinar un código simple.
Ir mostrando los primeros tipos de métodos en un código ya hecho. Constructores, propiedades consultables,
modificables, propiedades básicas y propiedades derivadas.

Para usar este proyecto, al alumno se le dan los tipos Estudiante y ClaseLaboratorio hechos y debe escribir
código en el main de la clase Test(en el caso de Eclipse) o mediante comandos (caso BlueJ) para crear objetos 
de tipo Estudiante, crear un objeto de tipo ClaseLaboratorio, inscribir a los alumnos en el laboratorio mediante
el uso del método inscribirEstudiante, mostrar los objetos tras realizar las disintas operaciones.
